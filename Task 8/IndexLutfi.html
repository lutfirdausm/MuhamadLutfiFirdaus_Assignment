Task 8, Ini adalah calon website dengan menerapkan struktur HTML.
HTML (Hyper Text Markup Language) merupakan struktur dasar atau kerangka dari halaman web untuk mendeskripsikan isi halaman web. HTML bukan bahasa pemrograman, melainkan bahasa markup yang menggunakan serangkaian kode/tanda (biasa disebut tag ), untuk memerintahkan web browser untuk menampilkan teks, gambar, tautan, dan multimedia lainnya. HTML adalah dasar utama dari hampir semua situs web di internet.
Aturan pada HTML antara lain:
  1. Penulisan tag di HTML tidak sensitif, tidak masalah memakai huruf besar atau kecil, Namun sangat disarankan menggunakan huruf kecil semua.
  2. Penulisan tag harus ditulis dengan Benar (Opening dan Closing Tag).
  3. Penulisan atribut menggunakan Tanda Petik (" " atau ' ').
  4. Penulisan elemen harus Tersusun Rapih (Nesting).
  5. Menggunakan elemen Sesuai Fungsinya.

Tag HTML Kelompok
Tag kelompok digunakan untuk mengelompokkan konten agar struktur halaman lebih jelas, rapih, mudah dibaca, ramah, serta aksesibilitas.

Tag Fungsi Utama
< main > Konten utama halaman
< section > Bagian berdasarkan topik
< header > Bagian atas/ Judul
< footer > Bagian Bawah/ Penutup
< nav > Menu Navigasi
< div > Wadah umum untuk layout
